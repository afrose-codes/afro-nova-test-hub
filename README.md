# 🧠 Afro Nova Test Hub

An online examination system developed to conduct tests efficiently with admin and user functionalities.

## 🚀 Features

- Admin login and dashboard
- Online exam system
- Result generation
- User registration
- Download mark list

---

## 🛠️ Technologies Used

- PHP
- MySQL
- HTML
- CSS
- JavaScript
- XAMPP

## ⚙️ How to Run

1. Install XAMPP  
2. Place project in `htdocs`  
3. Start Apache & MySQL  
4. Open browser:  
   http://localhost/afro-nova-test-hub
